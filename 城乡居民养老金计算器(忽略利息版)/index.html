计算说明：

本计算器计算方式：

1. 个人账户总额 = 所有缴费金额 × 缴纳年数之和

2. 个人账户养老金 = 个人账户总额 ÷ 139（城乡居民养老保险计发月数）

3. 基础养老金 = 当地政府规定的基础养老金标准

4. 每月养老金总额 = 个人账户养老金 + 基础养老金

官方实际计算方式（参考）：

1. 个人账户总额 = 缴费金额 × 缴纳年数 + 利息（由国家统一规定利率）

2. 个人账户养老金 = 个人账户总额 ÷ 139

3. 基础养老金 = 当地政府规定的基础养老金标准（可能随政策调整）

4. 每月养老金总额 = 个人账户养老金 + 基础养老金
