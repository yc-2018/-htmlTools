这版的实现思路

按目录拆工具

不是把很多属性挤在一个页面里，而是像原站一样拆成独立条目，切换更清晰。

一改即看

每个工具都带自己的预览方式，修改参数后会立刻反映到结果区。

代码可直接复制

CSS 页输出样式片段，HTML 页输出标签片段，方便直接贴进项目或文档。

后续还能继续补
