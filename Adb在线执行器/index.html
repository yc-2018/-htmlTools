Guide

使用前准备

  • 手机需打开开发者选项，并启用 USB 调试。
  • 首次连接时，需要在手机上确认“允许 USB 调试”。
  • Windows 通常还需要安装对应厂商 USB 驱动。
  • 浏览器建议使用最新版 Chrome 或 Edge，Safari / Firefox 通常不支持 WebUSB。
适合做什么
激活 Shizuku、执行 pm / am / settings / getprop 等常用非交互命令。
不适合做什么
需要持续交互输入、需要终端控制、或依赖复杂 TTY 行为的命令。
